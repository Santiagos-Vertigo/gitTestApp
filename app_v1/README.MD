# Pandas Data Exploration – App V1

## Purpose
Basic data exploration using Pandas:
- Load CSV / JSON / TXT / XML files
- Inspect structure and types
- Prepare data for downstream analysis or storage

## Structure
- `src/` – Pandas scripts
- `data/raw/` – Source datasets
- `data/processed/` – Cleaned outputs
- `requirements.txt` – Python dependencies

## Usage
```bash
python src/01_pandas.py
